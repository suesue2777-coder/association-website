# 日本特定技能教育支援協会 公式ホームページ制作コンテキスト

最終更新日: 2026年6月30日

## 1. プロジェクト概要

一般社団法人 日本特定技能教育支援協会の公式ホームページを制作するプロジェクトです。

主な閲覧対象は、日本企業、自治体、政府・行政関係者、教育機関、海外の連携機関です。協会の公共性、透明性、教育性及び国際性を伝え、会員参加、教育連携、海外連携及び問い合わせにつなげます。

現在のサイトは `site/` 以下にある HTML / CSS / JavaScript 形式の静的サイトです。サーバーなしでも閲覧でき、将来はCMSやフォーム送信機能を追加できる構造を想定しています。

## 2. 団体の位置づけ

- 海外人材の教育・育成を支援する非営利・教育支援団体
- 特定技能制度の適正運用に資する教育体制及び制度を構築する団体
- 海外政府、行政機関、教育機関、日本企業及び自治体との国際連携を推進する団体
- 人材紹介会社又は職業紹介事業者として表現しない
- 登録支援機関として表現しない
- 採用あっせんではなく、教育支援、制度構築及び国際連携を担うことを明記する

## 3. 正式名称と表記ルール

- 日本語名: 一般社団法人 日本特定技能教育支援協会
- 英語名: `JAPAN SPECFIED SKILLED EDUCATION SUPPORT ASSOCIATION`
- 英語名は現在、依頼者指定の `SPECFIED` 表記を使用しているため、明示的な変更指示があるまで自動修正しない
- 文体は行政・企業向けの簡潔で誠実な表現とし、実績や連携状況を誇張しない
- 未締結のMOU、未確定の提携先及び未実施の事業を確定事項として掲載しない
- `AHDF` 及び `Asia Human Development Foundation` はサイトから削除済みであり、再掲載しない

## 4. 活動内容

- 特定技能教育支援
- 日本語教育支援
- 技能教育支援
- 安全教育
- 海外政府・教育機関との連携
- MOU締結支援
- 日本企業向け人材育成支援
- 海外候補者の教育・育成体制構築
- 教育支援制度の構築
- 育成就労管理団体、登録支援機関、就労実施企業等に対する外国人管理教育

外国人管理教育には、外国人材の適正な管理、教育、生活支援及び法令遵守に関する研修支援を含みます。

## 5. 海外ネットワーク

現在の対象国は次の4か国です。

1. バングラデシュ
2. ネパール
3. インドネシア
4. カンボジア

各国について、連携目的、教育・育成分野、現地パートナー及び今後の展開を掲載します。正式な現地パートナー名は、提携状況と公開承諾を確認してから掲載します。

- `site/index.html`: 日本語トップページ

- `site/privacy.html`: 個人情報保護方針
- `site/terms.html`: サイト利用規約
- `site/disclosure.html`: 情報公開方針
- `site/404.html`: 404ページ
- `site/search.html`: サイト内検索
- `site/index-en.html`: 英語トップページ
- `site/index-bn.html`: ベンガル語トップページ
- `site/index-ne.html`: ネパール語トップページ

共通ナビゲーションの順序は次のとおりです。

`トップ / 協会概要 / 事業内容 / 海外ネットワーク / 会員名簿 / 会員制度 / お知らせ / お問い合わせ`

## 7. 多言語対応

対応言語は、日本語、英語、ベンガル語、ネパール語、インドネシア語、ベトナム語及びカンボジア語です。

現時点では各言語のトップページを用意しています。下層ページは日本語が中心です。翻訳を追加する際は、固有名詞、制度名及び法的な表現についてネイティブ又は専門家による確認を前提とします。

## 8. デザイン方針

- 国際機関・官公庁サイトに近い、硬く信頼感のあるデザイン
- 青、白、薄いグレーを基調とする
- 清潔感、公共性、教育支援及び国際連携を感じる構成
- 高齢者や行政関係者にも読みやすい文字サイズと明確な階層
- PC、タブレット及びスマートフォンに対応
- 固定ヘッダー、モバイル用メニュー、言語切替、文字サイズ及び配色切替を備える
- 控えめな動きに限定し、装飾より情報の読みやすさを優先する
- 正式ロゴは `site/assets/official-logo-full.png`、ヘッダー用マークは `site/assets/official-logo-mark.png`

## 9. 会員制度と会員名簿

会員区分は次の4種類です。

- 正会員
- 賛助会員
- 特別会員
- 海外パートナー会員

会員名簿は会員制度ページ内ではなく、独立した `site/members.html` に掲載します。会員番号、会員氏名、住所、電話番号、メールアドレス及びホームページURLを表示でき、会員名又はURLから会員サイトへ移動できる構造です。

個人情報は、本人又は団体から公開承諾を得た項目だけを掲載します。未承諾の住所、電話番号及びメールアドレスは公開しません。会員データは `site/member-directory.js`、入力用ひな型は `site/member-list-template.csv` です。

## 10. 外部機関リンク

フッターには、関係する公的機関の公式サイトへのロゴ付きリンクを設置しています。

- 出入国在留管理庁
- 外務省
- 厚生労働省
- 環境省
- 農林水産省
- 国土交通省

リンク先及びロゴを変更する場合は、各機関の最新の公式URLとロゴ利用条件を確認します。

## 11. 現在の未確定・仮情報

次の項目は正式情報への差し替えが必要です。

- 主たる事務所の正式住所
- 代表者氏名及び役職
- 設立年月日
- 法人番号
- 正式な電話番号
- 正式な問い合わせメールアドレス
- 代表挨拶の確定原稿
- 公開可能な会員情報
- 海外の正式提携先及びMOU情報
- 実際のお知らせ、活動報告及びセミナー情報
- 問い合わせフォームの送信先と個人情報保護方針

正式法人情報は `site/site-config.js` で一元管理します。団体名、英語名、代表者、設立年月日、法人番号、所在地、電話番号、メールアドレス、問い合わせ送信先及び公開ドメインは、このファイルから差し替えます。現在の「正式情報未設定」は公開前に確定情報へ変更します。

## 12. 更新時の技術ルール

- 共通ヘッダー、フッター、ナビゲーション及び言語情報は `site/app.js` で管理する
- 翻訳トップページ本文は `site/home-i18n.js` で管理する
- 共通デザインは `site/styles.css` で管理する
- 公開前の仮お知らせは `site/site-config.js` の `content` で管理し、将来的にJSON又はCMSへ移行できる
- 問い合わせフォームは送信エンドポイント又は正式メールアドレスの設定後に利用できる
- 問い合わせ方式は `site/site-config.js` の `contact.deliveryMode` で `disabled`、`mailto`、`api` を切り替える
- 法務ページの制定日は `site/site-config.js` の `legal` で一元管理する
- 公開承認状態は `site/site-config.js` の `publication` で管理し、`site/publication-status.html` で確認する
- `site/generate-publication-report.mjs` でプロジェクト直下に `PUBLICATION_FINAL_REPORT.md` を生成する
- SEOメタ情報は `site/generate-meta.mjs` が `site/site-config.js` の公開ドメインと団体名を参照して生成する
- 画像は `site/assets/` に保存し、差し替え可能なファイル構成を維持する
- ページ追加やナビゲーション変更時は、全言語のラベルとモバイル表示も同時に確認する

## 13. 公開前チェック

- 正式名称、住所、代表者、連絡先に誤りがない
- デモ用文章、仮日付及び仮連絡先が残っていない
- 未確定の連携や実績を確定事項として記載していない
- 会員情報の公開承諾を確認している
- PC、タブレット及びスマートフォンで表示が崩れない
- ナビゲーション、言語切替及び外部リンクが動作する
- 問い合わせフォームに必要な項目が揃っている
- 個人情報保護方針、利用規約等の必要性を確認している
- 翻訳内容を各言語の確認者が校正している
- 官公庁ロゴの利用条件及びリンク先を確認している

## 14. ローカル確認方法

`site/index.html` をブラウザで開くか、次のコマンドでローカルサーバーを起動します。

```bash
cd site
python3 -m http.server 8080
```

確認URL:

`http://localhost:8080/`
