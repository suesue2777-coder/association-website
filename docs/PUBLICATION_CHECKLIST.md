- [ ] `site/site-config.js` の `contact.deliveryMode` を確定

- [ ] `site-config.js` にAPIキー、認証トークン等の秘密情報を記載していない
- [ ] 送信APIがJSON形式のPOSTを受信できることを確認
- [ ] 送信成功時に `contact-complete.html` へ遷移することを確認
- [ ] 送信失敗時のエラー表示を確認
- [ ] 正式メールアドレスへの通知を確認
- [ ] 迷惑送信対策、保存期間、アクセス権限を確認

送信APIを使わずメールアプリ方式とする場合は、`contact.deliveryMode` を
`mailto` にして `contact.email` を設定します。
